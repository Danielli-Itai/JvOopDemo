# JvOopDemo

# Usage

### To run the test for specific project
Copy the jar to your project
Use the following application arguments: <b> run_workspace <your workspace root folder> </b>

### To run the test for your entire workspace 
Use the following application arguments: <b> run_workspace <your workspace root folder> </b>
